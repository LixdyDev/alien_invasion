# Contributing to Alien Invasion

First off, thank you for considering contributing to this project! It's people like you that make open-source such a great community.

## How to Contribute

### 1. Find an Issue
Look through the existing issues to see if there is a bug or feature you want to tackle. If you have a new idea (like a new alien type or power-up), feel free to open a new issue to discuss it first!

### 2. Make Your Changes
1. Fork the repository.
2. Create a new branch for your feature (`git checkout -b feature/YourFeatureName`).
3. Make your changes and test them locally.
4. Ensure your code is clean and well-commented.

### 3. Submit a Pull Request
1. Push your branch to your fork (`git push origin feature/YourFeatureName`).
2. Open a Pull Request on the main repository.
3. Describe your changes in detail so they can be reviewed quickly.
