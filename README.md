# Alien Invasion 👾

> **Status:** 🚧 Work in Progress 

## About This Project
This is a 2D space shooter game built using Python and the Pygame library. 

I am currently developing this project step-by-step by following the hands-on project section of **Python Crash Course (3rd Edition)** by Eric Matthes. The goal of this project is to practice Object-Oriented Programming (OOP) principles, event handling, and project organization in Python.

## Current Progress
* [x] Set up basic Pygame display window
* [x] Configured main game loop and event listening
* [x] Created `Settings` and `Ship` classes
* [ ] Add ship movement
* [ ] Add firing mechanics (bullets)
* [ ] Spawn alien fleet
