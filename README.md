# Alien Invasion 👾

> **Status:** 🚀 Base Game Complete | 🤝 Open for Contributions!

## About This Project
This is a 2D space shooter game built using Python and the Pygame library. 

I initially developed this project step-by-step by following the hands-on project section of **Python Crash Course (3rd Edition)**. The core mechanics—including full ship movement, firing projectiles, alien fleet generation, and collision detection—are fully implemented and functioning.

Instead of keeping this as a solo project, **I am officially opening this repository to the community!** If you are learning Pygame, Python, or just want to practice contributing to open-source projects, you are invited to fork this game and add your own creative spin to it.

## 🛠️ Current Features (What's Already Built)
* [x] Smooth, continuous 4-way ship movement
* [x] Spacebar firing mechanics with memory-optimized bullet deletion
* [x] Automated alien fleet generation (grid layout)
* [x] Fleet movement, edge-detection, and dropping mechanics
* [x] Full collision detection (Bullets hit Aliens, Aliens hit Ship/Bottom)
* [x] Fullscreen mode and quick-quit ('Q') shortcuts
* [x] Game Over states and ship resetting

## 🌟 Calling All Pygame Learners! (How You Can Help)
The base game is solid, but it needs your creativity! Here are some ideas for features you can build and submit:

* **Leveling System:** Increase alien speed or fleet size when a wave is cleared.
* **Scoring System:** Keep track of points, high scores, and display them on the screen.
* **Power-ups:** Add shields, spread-shot lasers, or speed boosts.
* **New Enemies:** Create different types of aliens that require multiple hits or shoot back!
* **Sound Effects & Music:** Bring the game to life with audio files.
* **Main Menu:** A starting screen with a "Play" button.

### How to Contribute
1. **Fork** this repository.
2. **Clone** it to your local machine.
3. Create a new branch for your feature (`git checkout -b feature/AmazingNewFeature`).
4. Commit your changes (`git commit -m 'Add some AmazingNewFeature'`).
5. Push to the branch (`git push origin feature/AmazingNewFeature`).
6. Open a **Pull Request** and let's review your awesome code!
